PlayerSDK

Introduction

By using PlayerSDK you can add to your iOS application the ability to play panoramic video from our 360 camera. Player supports playback two types formats of panoramic videos: raw video from camera and stitched video in equirectangular projection.

PlayerSDK can playback live video streams and files in specified formats. With help of the PlayerSDK, you can record a camera raw format to a client device or take snapshots. Also has a feature supports audio transmission from the client device to the camera.

Installation

Embedded Framework

  • Open the PlayerSDK folder, and drag the PlayerSDK.framework into the Project Navigator of your application’s Xcode project below project icon.
  • In the appeared window uncheck Add to targets.
  • Select your project icon in Project Navigation tree. In the tab bar at the top of that window, open the General panel.
  • Click on the + button under the Embedded Binaries section.
  • Select the PlayerSDK.framework.
  • And that’s it!

The PlayerSDK.framework is automagically added as a target dependency, linked framework and embedded framework in a copy files build phase which is all you need to build on the simulator and a device.

Example Code

Import PlayerSDK:

#import <PlayerSDK/PlayerSDK.h>

Declare in your class interface:

@property (strong, nonatomic) PlayerSDK360_PlayerView *playerView;
@property (assign, nonatomic) CGFloat zoom;

Init player and prepare gestures:

- (void)viewDidLoad
{
    [super viewDidLoad];

    // Prepare frame for player viewport
    CGRect screenBounds = [UIScreen.mainScreen bounds];

    // Init player
    self.playerView = [[PlayerSDK360_PlayerView alloc] initWithFrame:screenBounds];

    // Add player view to root controller view
    [self.view addSubview:self.playerView];

    // Setup pinch gesture for zoom viewport
    self.zoom = 1.0;

    UIPinchGestureRecognizer *pinch 
    = [[UIPinchGestureRecognizer alloc] initWithTarget:self
                                                action:@selector(handlePinch:)];
    [self.playerView addGestureRecognizer:pinch];

    // Play video
    [self.playerView openUrl:VIDEO_PATH
             equirectangular:true
               autoReconnect:true];
    [self.playerView play];
}

Handle pan gesture for camera rotation:

- (void)touchesMoved:(NSSet *)touches withEvent:(UIEvent *)event
{
    UITouch *touch = [touches anyObject];
    CGPoint location = [[touches anyObject] locationInView:self.view];
    CGPoint lastLocation = [touch previousLocationInView:self.view];

    [self playerView].yawAngle += (lastLocation.x - location.x) * 0.2;
    [self playerView].pitchAngle += (lastLocation.y - location.y) * 0.2;
}

Handle pinch gesture for zooming:

- (void)handlePinch:(UIPinchGestureRecognizer *)gestureRecognizer
{
    float zoom = self.zoom * gestureRecognizer.scale;
    if (zoom > 2.0f) {
        zoom = 2.0f;
    }
    else if (zoom < 0.8f) {
        zoom = 0.8f;
    }

    if (gestureRecognizer.state == UIGestureRecognizerStateEnded ||
        gestureRecognizer.state == UIGestureRecognizerStateCancelled ||
        gestureRecognizer.state == UIGestureRecognizerStateFailed) {
        self.zoom = zoom;
    }

    [self.playerView setScale:zoom];
}
